CS2 工具箱 - 一款专为 CS2 玩家打造的快捷话术发送工具

项目介绍
CS2 工具箱是一款基于 Python + Tkinter 开发的桌面应用，专为 CS2（Counter-Strike 2）玩家设计，提供快捷话术发送、自定义文本管理、一键开火喊话等核心功能。无需手动输入，通过按钮或快捷键即可快速发送预设文本，提升游戏内沟通效率，让你专注于对战操作！
核心功能
1. 快捷话术发送

    内置 5 条常用游戏话术（可自定义修改）
    点击文本按钮即可快速发送，支持中文无压力
    支持添加 / 删除自定义话术，满足个性化沟通需求

2. 一键开火喊话

    默认 3 条 CS2 专属开火话术：「冲就完事了！」「这波我来 Carry」「别白给！跟我上」
    支持顺序轮播 / 随机乱序两种模式切换
    可自定义开火话术列表，支持多文本批量设置

3. 文本管理功能

    支持单条添加自定义文本，实时生效
    支持 TXT 文件批量导入词汇，快速扩充话术库
    文本自动保存，下次启动自动加载历史数据

4. 便捷快捷键操作

    F1：显示 / 隐藏窗口（游戏内不遮挡视野）
    F2：一键触发开火喊话（游戏内快速调用）
    ESC：退出程序（安全关闭，数据自动保存）

5. 个人博客直达

    内置「我的博客」快捷按钮，一键访问 chukai.cc
    弹窗提示「欢迎来到初开的个人博客」，提升用户体验

技术栈

    开发语言：Python 3.x
    GUI 框架：Tkinter（原生组件，轻量无依赖）
    核心依赖：pyperclip（剪贴板操作）、Pillow（图片处理，已移除背景功能）、webbrowser（浏览器调用）
    打包工具：PyInstaller（支持 Windows 平台一键打包为 EXE）

下载与使用
1. 直接下载

    前往 GitHub Releases 页面下载最新版 EXE 文件：GitHub Releases（替换为你的仓库地址）
    无需安装，下载后右键「以管理员身份运行」即可使用

2. 源码运行
环境准备
bash
运行

# 克隆仓库
git clone https://github.com/你的用户名/cs2-toolbox.git
cd cs2-toolbox

# 安装依赖
pip install pyperclip pillow -i https://pypi.tuna.tsinghua.edu.cn/simple

运行程序
bash
运行

python cs2_toolbox.py

3. 打包为 EXE（可选）
bash
运行

# 安装打包工具
pip install pyinstaller

# 打包命令（带图标，替换为你的图标路径）
pyinstaller -F -w --icon=你的图标.ico --noupx cs2_toolbox.py

注意事项

    程序需要管理员权限运行，否则可能无法正常发送文本
    游戏内使用时，确保 CS2 窗口处于前台状态
    自定义文本支持中文、英文等多种字符，无长度限制
    话术数据保存在程序目录下的 cs2_msg.txt 和 cs2_fire_config.txt，可手动编辑修改

开发背景
作为一名 CS2 爱好者，在游戏中经常需要快速与队友沟通，但手动输入文本既耗时又影响操作。因此开发了这款工具箱，通过预设话术和快捷键，让沟通更高效。同时保留了简洁美观的 UI 设计，兼顾实用性和视觉体验。
